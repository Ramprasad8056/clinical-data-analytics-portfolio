# Bellabeat Smart Device Case Study

## Objective
Analyze smart device usage data to identify activity trends and provide marketing recommendations for Bellabeat.

## Tools Used
- Excel
- SQL
- R (tidyverse, ggplot2)

## Key Insights
- Higher activity leads to higher calorie burn
- Many users have prolonged sedentary time
- Personalized nudges can improve engagement

## Outcome
Business-focused recommendations to enhance product usage and marketing strategy.
